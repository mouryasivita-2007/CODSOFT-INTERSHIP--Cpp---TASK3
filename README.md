**Overview of the Project**
**Project**: Tic-Tac-Toe Game (C++)
**Objective**: The objective of this project is to implement a two-player Tic-Tac-Toe game using C++. The game allows players to select positions on a 3×3 board and automatically checks for winning conditions, draws, and invalid moves. This project reinforces logic building, flow control, and array handling in C++.

**Key Activities**
1)Board Setup: Representing the 3×3 grid for gameplay.
2)Player Input: Taking moves from Player 1 and Player 2.
3)Game Logic: Checking for winning patterns and draws.
4)Turn Switching: Alternating turns between two players.
5)Result Display: Declaring the winner or a draw.

**Technologies Used**
1)C++: Game implementation and logic control.
2)Dev-C++: IDE used for compiling, debugging, and testing the project.

**Features**
1)Classic two-player board game.
2)Displays board after every move.
3)Detects win patterns such as rows, columns, and diagonals.
4)Handles invalid inputs during gameplay.
5)Beginner-friendly and extendable.
